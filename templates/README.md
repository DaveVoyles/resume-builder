# 📝 Templates

Reusable, blank starter files — safe to commit because they carry no candidate data, just
section headers and prompts to fill in. No subfolders here.

| Template | Used by |
| --- | --- |
| [`candidate-intake.md`](candidate-intake.md) | Scaffolded into every new workspace as `candidate/inputs/notes/intake.md` by `npm run setup` / `init`. A blank set of questions (basic info, target roles, work history) for the candidate to answer before or during [grill intake](../docs/playbooks/grill.md). |
